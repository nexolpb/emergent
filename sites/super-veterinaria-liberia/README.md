# Super Veterinaria Liberia (Emergent Export)

This folder contains the exported landing page project generated in Emergent AI.

## Local development

```bash
cd sites/super-veterinaria-liberia/frontend
npm install
npm start
```

## Production build

```bash
cd sites/super-veterinaria-liberia/frontend
npm run build
```

## Notes
- CRA + CRACO + Tailwind.
