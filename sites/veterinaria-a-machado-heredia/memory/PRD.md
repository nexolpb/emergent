# PRD: Clínica Veterinaria Machado - Landing Page Preview

## Project Overview
**Client:** Nexo Sites (sales preview for Clínica Veterinaria Machado)
**Type:** Static 1-page landing page
**Location:** San Pablo, Heredia, Costa Rica
**Date Created:** December 2025

## User Personas
1. **Nexo Sites Sales Team** - Uses this preview to demonstrate website capabilities to potential clients
2. **Prospective Website Buyers** - Veterinary clinic owners evaluating website services

## Core Requirements (Static)
- ✅ Single-page static React app
- ✅ Spanish neutral (no voseo)
- ✅ All CTAs → Nexo Sites WhatsApp (wa.me/50689939071)
- ✅ Business contact info = display only
- ✅ No fake testimonials
- ✅ Mobile-first responsive design
- ✅ Pricing section with dark background (#022C22)

## Sections Implemented
1. **Hero** - Headline, trust bullets (4.3★, Dr. Machado, location), dual CTAs
2. **Services** - 9 service cards (Consulta, Radiología, Cirugía, etc.)
3. **Why Choose Us** - 4 feature cards (experience, equipment, care, specialists)
4. **Location** - Google Maps iframe embed
5. **Contact** - Phones, email, hours display
6. **Pricing Teaser** - Dark background, $349 starting price, WhatsApp CTA
7. **Footer** - Nexo Sites branding
8. **Floating WhatsApp** - Sticky button on mobile
